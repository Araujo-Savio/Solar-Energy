# Solar Energy

`Análise e desenvolvimento de sistemas`

`Projeto: Desenvolvimento de uma Aplicação Interativa`

`Eixo 2 - Segundo Semestre`

Este projeto tem como objetivo conectar usuários e empresas do setor de energia solar, facilitando o acesso a informações, serviços e orçamentos.
A aplicação busca resolver a dificuldade que muitas pessoas enfrentam ao procurar empresas confiáveis que atendam sua região, promovendo comparação de propostas, transparência e segurança no processo de contratação.
